## Testing
For testing you only need a web-server. You can install the "Chrome web server" addon.

|    |            |            | 
|----------|:-------------:|:-------------:|

- Install chrome server addon
- Go to advanced options, generate certificate and enable https.
- Select folder, start server 
- Open the https url
